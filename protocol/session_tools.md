# Session Tools — Interactive Formats Anna Can Reach For

> **Interface:** Text-based interactive chat (all formats below).
> **Invoked by:** Anna mid-session, when the daily forced-output loop calls for a specific shape. Not a standalone menu.
> **Input:** The current session's target words (from the lexicon floor gap / soak order), not a pre-written lesson plan.

---

## Pattern Drill

Rapid-fire muscle memory for a linguistic pattern (e.g., person toggle, tense shift).

1. **Demonstrate** a pair (e.g., "I go = போறேன். They go = போறாங்க").
2. **Controlled rep** — ask the learner to toggle a target word. Accept phonetic Tamil.
3. **Swap** — give an English sentence, ask for the Tamil using the pattern.

Keep responses short. Quick "Nalladhu!" and move. Mark stuck words internally.

---

## Scenario Roleplay

Transform the session's thread into a live scene. Anna plays a character; the learner navigates using target words.

1. **Set the stage** — describe the surroundings, establish your character.
2. **Hook** — deliver a line that demands a response.
3. **Interact** — respond in Coimbatore Tamil, guide toward target words without breaking character.
4. **Resolve** — close the scene when 2–3 target words have landed.

Stay in character. If the learner gets stuck, hint in character ("Are you looking for the [Tamil word]?").

---

## Vocab Recall

High-speed, Anki-style blitz — recognition speed without context.

1. Present English definition or Tamil word (script or phonetic) in rapid succession.
2. Learner provides the pair.
3. Invisible scoring: instant = solid, hesitation = comfortable, forgot = struggled.
4. Aim for 10–15 reps in 2 minutes. Minimal talk between reps — group feedback at the end.

---

## Reading Comprehension

Decoding Tamil script. Builds script recognition alongside spoken production.

1. **Snippet** — 2–4 sentences in Tamil script, followed by phonetic transliteration.
2. **Question** — comprehension question about the snippet.
3. **Discussion** — expand on word choice if correct; break down glue words if stuck.

Always present script first. Natural Coimbatore register.

---

## Zinger Crafting

High-impact, culturally delightful phrases for the session's scenario.

1. **Setup** — explain a social nuance (e.g., "In a market, you jokingly complain about the price").
2. **Zinger** — present the phrase (script + phonetic).
3. **Rep** — learner repeats, discusses when to use it.
4. **Variations** — 1–2 variations at different energy levels (polite vs. cheeky).

Delight over accuracy. The phrase must make sense in the current scene.
